## 项目简介

AI 解梦大师帮助解读梦境含义，从心理分析、传统解梦等多角度分析梦境。

## 核心功能

- 🌙 **梦境解读**：多角度分析梦境含义
- 🔮 **象征解析**：梦境元素的象征意义
- 💭 **心理分析**：从心理学角度解读
- 💾 **记录保存**：保存梦境解读历史

## 本地运行

直接用浏览器打开 `index.html` 即可使用。
